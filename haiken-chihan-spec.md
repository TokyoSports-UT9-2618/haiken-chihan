# 廃県置藩シミュレーター｜開発仕様書

**バージョン:** 1.0  
**対象エンジニア:** Claude Code  
**最終更新:** 2026年3月  

---

## 1. プロジェクト概要

### 1.1 コンセプト

「廃県置藩」をテーマにしたブラウザゲーム。  
現在の市町村の白地図をベースに、プレイヤーが市町村をグループ化して「新しい藩（基礎自治体）」を設計するシミュレーター。

**ゲームの目的：** 県内のすべての市町村を「人口40万人±10万人」を目安に藩へ分割し切ること。

### 1.2 フェーズ構成

| フェーズ | 範囲 | 優先度 |    
|---|---|---|
| Phase 1（本仕様） | 福島県（59市町村）単体 | ✅ 今すぐ作る |
| Phase 2 | 東北6県ブロック | 将来 |
| Phase 3 | 全国 | 将来 |

## 3. ゲームルール・ロジック

### 3.1 基本ルール

| 項目 | 内容 |
|---|---|
| 対象 | 福島県59市町村（総人口 約188万人） |
| 目標 | 全市町村をいずれかの「藩」に割り当てること |
| 藩の人口目標 | **40万人（±10万人）** = 30万〜50万人がベスト |
| 想定藩数 | 4〜6藩 |

### 3.2 スコアリング

ゲームクリア（全市町村を藩に割り当て）時にスコアを計算する。

```
ベーススコア: 1000点

各藩のペナルティ:
  人口が30万〜50万の範囲内 → ペナルティなし（+ボーナス100点）
  人口が25万〜30万 or 50万〜55万 → -50点
  人口が25万未満 or 55万以上 → -200点

藩数ボーナス:
  4藩でクリア → +200点
  5藩でクリア → +300点（最高評価）
  6藩でクリア → +200点
  3藩以下 or 7藩以上 → +0点

最終スコア = ベーススコア + 各藩ボーナス - ペナルティ合計
```

### 3.3 操作フロー

```
1. 画面起動
   └─ 福島県の地図が表示される（全市町村が白/グレーで未割当状態）

2. 藩を新規作成
   └─ 「新しい藩を作る」ボタン → 藩の仮名（例: 藩A）が生成される
   └─ 現在の「編集中の藩」がハイライト表示

3. 市町村をクリック
   └─ 編集中の藩に市町村が追加される
   └─ 地図上でその市町村が藩のカラーに塗られる
   └─ サイドパネルの人口ゲージがリアルタイム更新

4. 藩を確定
   └─ 「この藩を確定する」ボタン
   └─ 藩に名前をつける（テキスト入力）
   └─ 確定後はその藩の市町村は固定色で表示

5. 手順2〜4を繰り返す

6. 全市町村が割り当て済みになったら
   └─ 「ゲームクリア！」表示
   └─ スコア計算・表示
   └─ 各藩のサマリー（名前・人口・市町村一覧）

7. リセットボタンで最初から
```

## 5. 実装上の注意点

### 5.1 GeoJSONフォールバック

Artifactがgeoshape.ex.nii.ac.jpへのfetchに失敗した場合、  
**CircleMarker（半径に人口比例）** で代替表示すること。

```javascript
// フォールバック例
MUNICIPALITIES.forEach(m => {
  const radius = Math.sqrt(m.pop) / 30;
  L.circleMarker([m.lat, m.lng], { radius, ... })
   .addTo(map)
   .on('click', () => handleMunicipalityClick(m.code));
});
```

### 5.2 状態管理

```javascript
const state = {
  municipalities: MUNICIPALITIES,  // マスター（不変）
  assignments: {},       // { "07201": "han_1", "07203": "han_2", ... }
  hans: [],              // [{ id, name, color, confirmed }, ...]
  currentHanId: null,    // 現在編集中の藩ID
  phase: "playing"       // "playing" | "completed"
};
```

### 5.3 クリック操作

- 未割当の市町村をクリック → 現在の藩に追加
- 編集中の藩に属する市町村をクリック → 藩から除外（トグル動作）
- 確定済みの藩の市町村はクリック不可（グレーアウトしてもよい）

### 5.4 レスポンシブ

- PC（1200px以上）: 地図7割・サイドパネル3割の横並び
- タブレット（768〜1199px）: 地図6割・サイドパネル4割
- スマホ（767px以下）: 地図上部・パネル下部の縦並び
